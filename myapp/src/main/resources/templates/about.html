About the App

This is just a prototype of the app Thaimei E-commerce. The goal is to build an e-commerce platform where shopping becomes easier, more efficient, and more automated.

Some Features of the App

  • User registration and login
  • Product listing and details
  • Shopping cart using session
  • Product management by sellers

Technologies Used

  • Java Spring Boot
  • Thymeleaf
  • MySQL
  • HTML, CSS
